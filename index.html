TinyURL + QR Workflow

Prompt -> ChatGPT URL -> TinyURL -> QR-Code

Prompt eingeben, URL automatisch erstellen, TinyURL erzeugen und QR-Code direkt kopieren oder als PNG speichern.

Generator

TinyURL Zugang

Der Token wird nur in der aktuellen Browser-Session gespeichert.

So erhalten Sie den TinyURL API-Token:

  1. TinyURL Konto erstellen oder anmelden: tinyurl.com/app
  2. Im Dashboard den Bereich Developer/API oeffnen.
  3. Einen neuen API-Token erzeugen und kopieren.
  4. Token in dieses Feld einfuegen.

API-Endpunkt: https://api.tinyurl.com/create

Prompt Eingabe

Nach Klick auf den Button werden Long-URL, TinyURL und QR-Code in einem Schritt erzeugt.

Ergebnisse

Prompt (URL-encoded)

Long-URL (ChatGPT)

TinyURL

QR-Code Vorschau

Ihr Browser unterstuetzt kein Canvas-Element.

Der QR-Code zeigt standardmaessig auf die TinyURL. Falls keine TinyURL erzeugt wurde, wird die Long-URL genutzt.

Datenschutz: Token wird nicht auf einem Server gespeichert.
